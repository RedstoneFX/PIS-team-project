# Вэб редактор табличной грамматика

### Описание
Данное приложение предназначено для редактирования "файлов грамматики" для другой программы, назначение которой - анализ и интерпретация таблиц EXCEL и т. п.

### Поддерживаемые платформы
Программа реализована на чистом html+css+js без использования фреймворков и может быть запущена в любой операционной системе и браузере.

### Как запустить
Вы можете воспользоваться любым способом открытия статичных страниц:
- открыть файл web/index.html непосредственно в браузере
- запустить вэб-сервер в папке web и подключится к нему
- использовать онлайн-хостинг для статических страниц
- открыть сайт https://redstonefx.github.io/PIS-team-project/

### Как редактировать
Для внесения изменений в код подойдет любой текстовый редактор, например, Visual Studio Code.
