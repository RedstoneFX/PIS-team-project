Распознает паттерн из "сырых" данных
#### Псевдокод
Создать новый пустой паттерн
Обновить `countInDocument` из данных, если там такое поле есть
Обновить `height` из данных, если там такое поле есть
Обновить `width` из данных, если там такое поле есть
Сохранить значение поля `style`, если такое поле есть (не терять неиспользуемые данные)
Вернуть созданный паттерн и флаг isRoot


#### Примечание
Использовать НЕСТАТИЧЕСКИЙ метод [[Range.fromString(strRange)]]
