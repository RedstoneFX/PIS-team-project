Создает объект "сырых" данных на основе текущего компонента
#### Псевдокод
Выкинуть ошибку, если у текущего паттерна нет типа
Создать новый пустой объект
Записать в объект общие данные паттерна...
	Записать в объект поле `style`, если `this.style` не пустой
	Записать поле `size`, если `width` и `height` не являются значениями по умолчанию
	Записать поле `description`, если описание не пустое
Дописать в объект данные, зависящие от типа, с помощью [[PatternExtension.serializeTo(rawData)]]
Вернуть созданный сериализованный объект
