Распознает значения позиции в указанных данных и обновляет текущую позицию
#### Псевдокод

Восстановить значения позиций по умолчанию с помощью [[Range.restoreDefault()]]
Если тип переданных данных - строка...
	Разбить строку на слова через запятую
	Для каждого слова...
		Удалить пробелы из слова
		Установить позицию по слову с помощью [[ComponentLocation.setPositionByWord(word, isInner)]]
Иначе если тип данных - массив...
	Для каждого элемента в массиве...
		Если элемент - строка со стороной...
			Установить позицию по слову с помощью [[ComponentLocation.setPositionByWord(word, isInner)]]
		Иначе если элемент - пара сторона-значение...
			Распознать интервал во втором элементе пары с помощью [[Range.fromString(strRange)]]
			Установить позицию по паре с помощью [[ComponentLocation.setPositionByWordAndRange(word, range)]]
Иначе если тип данных - объект...
	Для каждой пары ключ-значение...
		Распознать интервал во втором элементе пары с помощью [[Range.fromString(strRange)]]
		Установить позицию по паре с помощью [[ComponentLocation.setPositionByWordAndRange(word, range)]]
Иначе...
	Выбросить ошибку о непонятном способе задачи позиции
