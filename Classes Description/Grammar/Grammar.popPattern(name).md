Находит и извлекает паттерн из грамматики
#### Псевдокод
Вернуть null, если паттерна с таким именем нет
Извлечь паттерн из `#patterns`, если он там есть (его там больше не будет)
Если извлеченный паттерн является корневым...
	Вывести сообщение в консоль об удалении корневого паттерна
	Занулить `#rootPattern`
Вернуть извлеченный паттерн
